Как пользоваться БД

Образец списка статей

Начало работы с системой

  1. Любой посетитель может просматривать список сотрудников на главной странице и карточки отдельных сотруднков, находящиеся по ссылке «См.» в последней колонке списка сотрудников.
  2. Если посетитель является сотрудником и авторизовался в системе, то он может редактировать свою карточку, перейдя к странице редактирования по ссылке «Ред.» в последней колонке списка сотрудников.
  3. Если посетитель является сотрудником с правами доступа администратора, то он после авторизации в системе он(а) может редактировать по ссылке «Ред.» карточку любого сотрудника, а также редактировать формулу для подсчёта ПРНД по ссылке «ПЕРЕЧЕНЬ КОЛИЧЕСТВЕННЫХ ПОКАЗАТЕЛЕЙ РЕЗУЛЬТАТИВНОСТИ ТРУДА НАУЧНЫХ РАБОТНИКОВ», находящейся под списком сотрудников на гл. странице сайта БД.

Редактирование и просмотр карточки сотрудника

  1. Перейдя по ссылке Ред. на страницу редактирования карточки сотрудника, можно редактировать личные данные, загружать список статей по установленному образцу и вводить параметры для подсчёта ПРНД, который считается автоматически согласно установленной администратором формуле.
  2. Страница карточки сотрудника доступна с главной страницы по ссылке См. в последней колонке и ряду с именем сотрудника в таблице списка сотрудников или со страницы редактирования карточки сотрудника по ссылке Посмотреть данные. На странице карточки сотрудника можно посмотреть также список его публикаций или скачать публикации отдельным файлом Excel.

Загрузка файла Excel со списком статей

  1. Файл с публикациями должен быть создан в программе MS Excel / LibreOffice / OpenOffice в формате xls по образцу.
    Внимание: наименования колонок должны быть именно такими, как в образце.
  2. На страничке редактирования своей карточки в разделе «Публикации» выбрать Excel файл публикаций со своего локального диска, нажать кнопку «Submit». Публикации занесутся в базу данных и их можно будет посмотреть по ссылке «Публикации» как на странице редактирования, так и в разделе просмотра карточки сотрудника.

Подсчёт ПРНД

  1. Форма со списком параметров для подсчёта ПРНД вызывается по ссылке Пересчитать в разделе «ПРНД» на странице редактирования карточки сотрудника.
  2. Значения для каждого параметра вводится в соответствующем столбце формы, после чего внизу нажимается кнопка «Submit». ПРНД пересчитывается автоматически.

Переопределение формулы для подсчёта ПРНД

  1. Коэффициенты для подсчёта ПРНД могут меняться администрацией на этой странице, доступной с главной страницы системы по ссылке «Перечень количественных показателей результативности труда научных работников»
  2. Excel файл показателей для подсчёта ПРНД могут редактироваться администрацией по имеющемуся образцу Excel файла.
    Внимание: наименования колонок должны быть именно такими, как в образце.
  3. Изменение формулы для подсчёта ПРНД вместе с изменениями наименований параметров должно быть согласовано с наименованиями параментров в Excel файле показателей для подсчёта ПРНД.
  4. После выбора файла с показателями для подсчёта ПРНД и/или изменения формулы для подсчёта ПРНД надо нажать кнопку «Submit».
